引用

引用元素表示從其他來源引用的內容，可以選擇包含引用來源，引用來源必須位於 footer 或 cite 元素中，也可以選擇包含註解和縮寫等內嵌變更。

無來源的引用

您可以在引用中使用 Markdown 語法，例如 bold, italics, links, code。

程式碼區塊

使用反引號的程式碼區塊

<!doctype html>
<html lang="en">
<head>
 <meta charset="utf-8">
 <title>Example HTML5 Document</title>
</head>
<body>
 <p>Test</p>
</body>
</html>

縮排四個空格的程式碼區塊

<!doctype html>
